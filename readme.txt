YouCouch
--------
My first application based on Java for the Google AppEngine. Its very basic and uses the Youtube API to generate a random video feed on your interest.

Building YouCouch
-----------------
To build Youcouch use Netbeans IDE with the Google AppEngine Netbean Plugin and the Java Web Application Plugins installed. Then open the project and build the application.


Head over to http://youcouch.appspot.com/
